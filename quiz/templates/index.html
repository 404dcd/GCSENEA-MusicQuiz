Hello!

Welcome to my implementation of the OCR Computing GCSE NEA Project - the music quiz.

How do I play?

Just register for an account, then log in. You'll then be able to access the Play page - where you can start guessing away at song titles. For each song, you're provided with the artist name and the first letter of each word of the title. The game ends when you guess incorrectly twice - or complete every song in the database. :P

You gain 3 points for a correct guess first time, and 1 for a correct guess the second time. Once you finish a game, you'll be directed to the leaderboard where you can view the top highscores.

Pen-testing scope

This website, and any page, file or directory served by it. No attempts to DoS or DDoS please, and try to limit any damage caused in general. Prove command injection with a simple whoami, and SQL injection with a sleep command (though I am using prepared statements everywhere, good luck).

Details

This is a Python flask application, using Jinja templates and an sqlite3 database. I am running it on a small Azure server using the python waitress WSGI library.
